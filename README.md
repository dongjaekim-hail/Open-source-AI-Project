### Electra 모델 개요
Electra 모델은 자연어 처리 작업에 효과적이고 효율적인 최신 변형기반(transformer-based) 모델입니다. BERT와 같은 다른 모델들과 달리 사전 훈련 방식에서 독특한 접근 방식을 취합니다.

### 데이터 전처리
**파일:** `preprocessing.py`

이 스크립트는 훈련을 위한 데이터 준비를 담당합니다. 데이터를 정리하고, 정규화하며, Electra 모델에 적합한 형식으로 구조화하는 작업을 포함합니다.

### 설치 방법
이 프로젝트에 필요한 환경을 설정하려면 다음 명령어를 실행하세요:

```bash
pip install accelerate -U
pip install transformers[torch]
